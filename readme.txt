== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/spottr` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Spottr screen to configure the plugin

= How does the pluggin work? =
The plugin lets you post new or existing woocommerce products to Spottr in a few clicks. All that you have to do is download the plugin and sign in using your Spottr account. It imports Spottrs categories and tags and the set up is completed. Simply head on to create a new product and make sure to check the dialogue box under the product name field if you want to post it to Spottr.

For existing products, open your products page from your woocommerce admin and click “Sync to Spottr” beside each product name to post them to Spottr

= How do I manage orders? =
Orders are managed separately on woocommerce and on Spottr. Spottr just gives your more visibility to grow your sales

= Can I choose products to post to Spottr? =
Absolutely. You have the power to decide what to send to Spottr or not

= What information does the plugin collect? =
The plugin gets all your information from Spottr when you sign in. At the time of posting a product, the plugin also gets your physical location in order to tag the product to a location on Spottr. Not to worry, products that are posted to Spottr can be fully edited

= Can I edit products that I send to Spottr using the plugin? =
Absolutely. Products sent to Spottr using the plugin are fully editable

= Why the Spottr for woocommerce plugin? =
The plugin helps merchants to rapidly grow sales by posting new and existing products to the worlds most advanced marketplace that helps customers find anything for sale within 2 minutes of thinking about it
